Matrix CSV Export
How to export from Matrix 1. Go to Matrix → Search → Residential
2. Sold comps: Set Status = Sold · Date Sold = Last 12 months · Area = target zip codes · match property type
3. Select all results → Actions → Export → Spreadsheet (CSV)
4. Active listings: Repeat with Status = Active, same area
5. Combine both into one CSV file and upload below

Step 4 of 5
Review Adjustments
Each comp is automatically adjusted to your subject property using appraiser-style methodology. Set the condition for each comp — you know these properties better than the data does.
How this works Each comp is adjusted UP if it's inferior to your property (adds value) and DOWN if it's superior (subtracts value). The adjusted value is what each comp implies your home is worth. Condition is the only input you set — everything else is calculated from the data.
